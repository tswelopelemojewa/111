ABOUT

The Maximum Unsupported Span API is an application programming interface designed to enhance the safety within the mining Industry. This API makes use of multi-modal machine learning ensemble-algorithms (Classifiers and Regressors) to bring about an accurate prediction of the Maximum Unsupported Span (MUS) that will result in cost reduction, mining optimization and the overal improvement of safety thereby safe-gaurding the human lives.
